## 슬라이드별 발표 포인트

### 1. 타이틀
- 컨셉 한 줄: "애인처럼 여행 갈 때 짐을 야무지게 챙겨주는 서비스."
- 캐치프라이즈 "자기야, 그거 챙겼어?"로 시작.

### 2. 문제 ① — 짐 싸기는 큰일이다
- 시간이 오래 걸린다 / 그 나라 문화까지 알아야 한다
- 상황이 제각각: 미국 110V, 이슬람권 히잡
- 블로그를 수없이 뒤져도 꼭 빼먹는다.

### 3. 문제 ② — P도 J도 불편하다
- **P**: "짐 싸는 거 귀찮아, 한 번에 추천해줘"
- **J**: "신경 쓸 게 너무 많아, 다 찾아봐도 빠뜨려"
- 펀치라인: **"블로그 10번 찾아볼 시간에, 야무짐 한 번"**

### 4. 솔루션
- 단순 체크리스트 ❌ → **나 + 목적지 + 그날 날씨**의 함수.
- 3가지 가치: 날씨까지 / 네 취향대로 / 이유까지 설명.

### 5. 플로우 (4단계)
- 나이·성별 → 여행 취향 → 여행 정보 → 캐리어 추천
- 입력이 쌓일수록 추천이 정교해진다.

### 6. 취향
- 위생·준비성·더위/추위 인내·한식 5척도 + 이동/소비/계획 스타일 + 테마.

### 7. 결과 — 이유까지
- 캐리어 UI로 한눈에 (필수·권장·선택)
- 아이템마다 내 취향·여행 조건을 인용한 근거 제공.

### 8. 그 나라까지
- 여행 기간 날씨 요약 + 긴급 연락처·현지 문화 금기(제스처·사원·식문화).
- "짐만이 아니라 **그 나라**까지 챙긴다."

### 9~14. 페르소나 (클라이맥스)
> 핵심 메시지: **같은 서비스인데, 상황마다 전혀 다른 유니크템을 챙겨준다.**

**페르소나 1 — 방학 맞은 싸피생** · 📍 일본 후쿠오카 2박 3일
- 컨셉: 치안 좋은 근거리 + 즉흥·활동파
- 특이 성향: 위생 둔감(1), 즉흥적(1)
- 유니크템: 🩹 스포츠 테이핑 · 💊 소화제 · 🎒 최소한의 의류

**페르소나 2 — 꼼꼼한 20대 배낭여행객** · 📍 프랑스 파리 생존형
- 컨셉: 변수 많은 장거리 + 철저한 방어파
- 특이 성향: 위생 매우 민감(5), 만반의 준비(5)
- 유니크템: 📱 스마트폰 스트랩·옷핀 · 🚿 휴대용 샤워기 필터 · 🛡️ 베드버그 스프레이

**페르소나 3 — 30대 호캉스족** · 📍 태국 방콕 4박 5일
- 컨셉: 무더운 동남아 + 더위 취약 힐링파
- 특이 성향: 더위에 약함(1), 걷는 거 최소
- 유니크템: ❄️ 쿨링 시트 · 🌀 휴대용 선풍기 · 🧴 선크림

### 15. 추론 파이프라인
1. **데이터 수집** — 여행·취향·도시 태그 + 날씨·치안·건강·전압
2. **신호 태그화** — 5척도·날씨·문화를 가중치 태그로 변환 (예: 날씨 의류 태그 10, 선크림 7), 부모 태그 확장(서유럽→유럽)
3. **후보 카탈로그** — 아이템 점수화·필터링 → 상위 24개 선별
4. **LLM 랭킹** — 규칙북을 지식으로 주입 → 우선순위·근거 생성
5. **검증·병합** — 기본템 병합·중복제거, 실패 시 규칙 기반 폴백
6. **캐리어 출력** — 필수·권장·선택 + 이유까지 저장
- 핵심: **규칙북(Rulebook)을 LLM에 '지식'으로 주입** → 9개 영역 매핑 규칙을 종합·중복제거·우선순위화.

### 16. 기술 스택
- **Frontend**: Vue 3 · Vite · Pinia
- **Backend**: Django · DRF · JWT · 카카오 OAuth
- **AI**: SSAFY GMS 게이트웨이 (Claude·Gemini)
- **Weather**: OpenWeather + Open-Meteo(UV 보정)
- **Docker + Redis** (날씨 API 응답 캐싱) — 채택 이유:
  - **실시간성**: 일기예보 API의 최신 예보를 그대로 반영
  - **휘발성**: 예보는 계속 바뀌므로 3시간 TTL로 자동 만료
  - **호출 절감**: 같은 일정 재요청 시 외부 API 재호출 방지
  - 캐시 키 = trip · city · 출발일 · 도착일

### 17. ERD
- 데이터 모델 한 장. (User/Preference/TravelStyle · Country/City/태그·안전·건강 · Trip · Recommendation · Community)

### 18. 커뮤니티 + 클로징
- 내 짐 리스트를 공유하고 좋아요·댓글로 소통.
- 클로징: **"자기야, 그거 챙겼어? — 이제 야무짐이 챙겨줄게."**
