# 야무짐 추천 규칙북 (Packing Rulebook)

> LLM System 프롬프트에 "지식"으로 주입되는 매핑 규칙 모음.
> LLM은 유저/여행/목적지 데이터를 받아 아래 규칙을 **종합·중복제거·우선순위화**하여 준비물 JSON을 생성한다.

## 표기 규칙
- 양식: `조건 → 아이템 [시간축·공간축·목적축] (우선순위) "근거"`
- 시간축: 출발 전 / 이동 중 / 현지 / 귀국
- 공간축: 숙소 / 관광지 / 거리 / 이동
- 목적축: 안전 / 필수 / 편의 / 즐거움
- 우선순위: **필수 > 권장 > 선택**
- `[안내]` 표시는 챙기는 물건이 아니라 **주의/팁(notes)** 으로 별도 출력

## LLM 적용 원칙
1. 여러 규칙이 같은 아이템을 가리키면 **하나로 합치고**, 가장 강한 우선순위/근거를 채택한다.
2. `인원수`가 많으면 공용 아이템(멀티탭·상비약 등)은 **인원당 중복하지 않는다**.
3. 근거(`reason`)는 해당 유저의 취향/여행 조건을 **구체적으로 인용**해 작성한다.
4. `[안내]` 항목은 `items`가 아니라 `notes` 배열로 출력한다.

## TravelTag 운영 규칙

### 태그 계층
- 부모 태그는 기존 아이템 매칭 호환성을 위해 유지한다. 예: `유럽`, `수질 취약`, `치안`, `열대 기후`.
- 세부 태그는 도시/환경 정보를 더 정확히 표현할 때 사용한다. 예: `서유럽`, `석회수`, `소매치기 빈번`, `스콜`.
- 추천 엔진은 세부 태그를 부모 태그로 확장한다. 예: `서유럽`이 있으면 `유럽`도 신호로 본다.

### 현재 parent expansion
- `서유럽`, `동유럽`, `남유럽`, `북유럽` → `유럽`
- `수돗물 음용 불가`, `석회수` → `수질 취약`
- `소매치기 빈번`, `야간 치안 주의` → `치안`
- `고온다습` → `열대 기후`
- `강한 자외선` → `더위`
- `우기` → `열대 기후`, `상황대비`
- `스콜` → `우기`, `상황대비`
- `한랭 기후` → `추위`
- `큰 일교차` → `상황대비`
- `해양 환경` → `해변/해양`
- `산악 지형` → `등산/트레킹`

### 폐기 후보
- `미주`: `북미`와 `남미`로 분리한다. 기존 데이터 호환을 위해 즉시 삭제하지 않되, 신규 도시/아이템 태그에는 사용하지 않는다.

### 작성 원칙
- 도시는 가능한 한 세부 태그를 붙인다. 예: `파리`에는 `서유럽`, `소매치기 빈번`, `석회수`.
- 아이템은 넓은 부모 태그와 강한 세부 태그를 함께 둘 수 있다. 예: `RFID 지갑`에는 `치안`, `소매치기 빈번`.
- 여행 테마 태그와 환경/안전 태그는 같은 `City.travel_tags`에 저장하되, `tag_class`로 성격을 구분한다.

## 0. 기본 베이스 (조건 없이 항상)
- 여권(잔여 6개월↑) → 여권 [출발 전·이동·필수] (필수) "해외 신분증명"
- 항공권/e-티켓·숙소 바우처 → [출발 전·이동·필수] (필수)
- 해외결제 카드 + 현지통화 소액 현금 → [출발 전·이동·필수] (필수)
- 스마트폰·충전기·충전케이블 → [출발 전·이동·필수] (필수)
- 보조배터리(기내 휴대) → [출발 전·이동·편의] (권장) "위탁 불가, 기내 반입"
- 기본 상비약(진통제·소화제·밴드) → [출발 전·숙소·안전] (권장)
- 옷·속옷·양말(여행 일수 기반) → [출발 전·숙소·필수] (필수)

## 1. 숙소형태 (accommodation_type)
- 호텔/리조트/아파트 → 수영복·비치타월·방수파우치 [현지·숙소·즐거움] (선택) "수영장/스파/해변"
- 호텔/리조트 → [안내] 어메니티(샴푸·칫솔) 대개 제공 → 세면도구 최소화 가능
- 게스트하우스/호스텔 → 자물쇠(다이얼/열쇠) [출발 전·숙소·안전] (권장) "공용 사물함 도난방지"
- 게스트하우스/호스텔 → 개인 세면도구·수건·슬리퍼 [출발 전·숙소·필수] (권장) "미제공 일반적"
- 게스트하우스/호스텔 → 이어플러그·안대 [현지·숙소·편의] (권장) "도미토리 소음/조명"
- 캡슐호텔 → 이어플러그·안대·간단 세면도구 [현지·숙소·편의] (권장)
- 아파트/에어비앤비 → 세탁세제·간단 조리도구 확인 [현지·숙소·편의] (선택) "장기 자취형"

## 2. 날씨/기온 (WEATHER: temp, 강수확률, 습도, UV, 풍속)
- 최고기온 ≥ 30℃ → 선크림·모자·선글라스 [현지·거리·편의] (권장) "폭염/자외선"
- 최고기온 ≥ 33℃ → 휴대용선풍기·쿨토시 [현지·거리·편의] (선택)
- 최저기온 ≤ 5℃ → 패딩·목도리·장갑·핫팩 [출발 전·거리·필수] (필수) "한파"
- 최저기온 5~15℃ → 경량패딩/가디건 [출발 전·거리·권장]
- 일교차 ≥ 10℃ → 얇은 겉옷(레이어드) [출발 전·거리·편의] (권장)
- 강수확률 ≥ 50% → 우산·방수 가방커버·여벌 양말 [출발 전·거리·편의] (권장)
- 강수확률 ≥ 70%(우기) → 우비·방수신발/아쿠아슈즈 [현지·거리·편의] (선택)
- 습도 ≥ 80% → 통풍 잘되는 옷·땀티·휴대용 제습제 [현지·숙소·편의] (선택) "꿉꿉함"
- UV지수 ≥ 8 → SPF50+ 선크림·립밤·양산 [출발 전·거리·안전] (권장) "적도/고산"
- 풍속 강함(해안/고산) → 바람막이 [현지·거리·편의] (선택)

## 3. 취향 척도 (USER_PREFERENCE 1~5)
- 위생민감도 ≥ 4 → 샤워필터·손소독제·물티슈 다량·변기시트커버 [출발 전·숙소·편의] (권장)
- 위생민감도 = 5 → 개인 베개커버·세탁비누·일회용 변기시트 [출발 전·숙소·편의] (선택)
- 상황대비성 ≥ 4 → 비상약키트 확장(지사제·항히스타민·종합감기약)·여행자보험증서 [출발 전·숙소·안전] (권장)
- 상황대비성 = 5 → 멀티툴·예비 안경·소형 응급키트 [출발 전·이동·안전] (선택)
- 더위인내 ≤ 2 → 휴대용선풍기·쿨링스프레이·땀흡수밴드 [현지·거리·편의] (권장)
- 추위인내 ≤ 2 → 핫팩·내복(겨울)/얇은 가디건(여름 냉방 대비) [출발 전·거리·편의] (권장) "실내 에어컨도 대비"
- 죽어도한식 ≥ 4 → 즉석밥·고추장튜브·라면·김·컵국 [출발 전·숙소·즐거움] (권장)
- 죽어도한식 = 5 → 휴대용 전기포트·수저세트·밑반찬 [출발 전·숙소·즐거움] (선택)

## 4. 관광스타일 (TRAVEL_STYLE)
### 방문장소유형 (다중)
- 맛집 → 위생손수건·소화제·맛집 오프라인 지도 [현지·관광지·즐거움] (선택)
- 자연경관 → 운동화·보조배터리·경량 삼각대 [현지·관광지·편의] (권장)
- 유원지 → 우비(놀이기구)·편한 신발·미니선풍기 [현지·관광지·편의] (선택)
- 유적지 → 긴바지·스카프·양산·편한 신발 [현지·관광지·필수] (권장) "사원/종교시설 복장 제한"
- 박물관/미술관 → 유선 이어폰(오디오가이드)·가벼운 가방 [현지·관광지·편의] (선택)
- 액티비티 → 방수팩·아쿠아슈즈·래시가드·액션캠 [현지·관광지·즐거움] (권장)
### 이동유형
- 워킹맨 → 기능성 운동화·압박양말·발가락밴드·보조배터리 [출발 전·거리·편의] (권장)
- 걷는거 최소 → 택시앱(Grab/Uber)·교통카드·편한 단화 [출발 전·이동·편의] (선택)
### 소비유형
- 기념품 → 접이식 보조가방·여분 캐리어 공간 [출발 전·이동·편의] (선택)
- 기념품 → 에어캡·지퍼백(파손방지 포장) [귀국·이동·편의] (선택)
- 기념품 → 기념품 예산용 현금/카드 한도 확보 [출발 전·이동·편의] (선택) + [안내] 면세 한도 초과 시 자진신고
- 가성비 → 텀블러·휴대용 식기 [현지·거리·편의] (선택) "물/식비 절약"
- 가성비 → 간식·즉석식품 [출발 전·숙소·편의] (선택)
- 가성비 → 현지 마트/교통 앱·장보기용 에코백 [출발 전·이동·편의] (선택)
- 가심비 → 미식/스파 예약 확인서 [출발 전·이동·편의] (선택)
- 가심비 → 드레스업 의상·구두(파인다이닝/호캉스) [출발 전·관광지·즐거움] (선택)
- 가심비 → [안내] 카드 한도/여분 현금·고가품 여행자보험 점검
### 준비성
- 즉흥적 → 현지 유심/eSIM·구글맵 오프라인·번역앱 [출발 전·이동·필수] (권장) "현지 조달 전제, 연결성 중요"
- 계획적 → 일정표 출력·예약 확인서·체크리스트 [출발 전·이동·편의] (선택)

## 5. 치안 (COUNTRY_SAFETY)
- 여행경보 ≥ 2(황색) → 복대·비상금 분산·여권 사본/클라우드 백업 [출발 전·이동·안전] (권장)
- 여행경보 ≥ 3(적색) → [안내] 여행 재검토 권고 + 비상연락체계 점검
- crime_index 높음 / safe_alone_night 낮음 → 도어스토퍼·가방 잠금장치 [현지·거리·안전] (권장) + [안내] 야간 단독 외출 자제
- 소매치기 다발 → 앞으로 메는 가방·지퍼 자물쇠·RFID 카드홀더 [현지·거리·안전] (권장)
- 항상 → 긴급연락처(현지 대사관·영사콜센터 +82-2-3210-0404) 저장 [출발 전·이동·안전] (필수)

## 6. 위생/건강 (COUNTRY_HEALTH)
- 식수 부적합 → 생수 구매 안내·휴대용 정수필터/정수알약 [현지·거리·안전] (권장) + [안내] 양치/얼음 주의
- 샤워필터 필수국 → 샤워헤드 필터·여행용 트리트먼트 [출발 전·숙소·편의] (선택)
- 풍토병 주의(말라리아/뎅기) → 모기기피제(DEET)·모기장·긴팔·말라리아약(병원 처방) [출발 전·거리·안전] (권장→필수)
- 황열 위험국 → 황열 예방접종증명서(옐로카드) [출발 전·이동·필수] (필수) "미소지 시 입국 거부"
- 위생 전반 → 손소독제·마스크·지사제·정로환 [출발 전·거리·안전] (권장)

## 7. 전압/문화 (COUNTRY: voltage/plug, CULTURAL_TABOO)
- 플러그타입 ≠ 한국(C/F형) → 멀티 어댑터(돼지코) [출발 전·이동·필수] (필수)
- 110V 지역(미국 등) → [안내] 220V 전용 기기(고데기 등) 변압기 필요 점검
- 기기 많음 → 멀티탭(3구) [출발 전·숙소·편의] (선택)
- 문화금기=노출 제한(이슬람/사원) → 긴팔·긴바지·스카프·어깨 가리개 [현지·관광지·필수] (권장)
- 문화금기=주류 제한 → [안내] 음주/주류 반입 제한 구역 주의
- 문화금기=제스처/신발 → [안내] 실내 탈화·특정 제스처 주의
- 팁 문화 존재 → 소액 지폐(팁용) [출발 전·거리·편의] (선택)

## 8. 동행/기간/현지어 (TRIP)
### 동행자 유형
- 가족(유아 포함) → 상비약·체온계·물티슈·간식·아기띠/유모차 [출발 전·숙소·필수] (권장)
- 가족(부모님) → 어르신 상비약·예비 안경·편한 신발 [출발 전·숙소·안전] (선택)
- 연인 → 커플 상비약·셀카봉/삼각대 [현지·관광지·즐거움] (선택)
- 친구/동행 구함 → 경비 정산앱·보조배터리 공유 [출발 전·이동·편의] (선택)
- 인원수 ≥ 4 → [안내] 공용 상비약·멀티탭은 1개로 통합
### 여행 기간 (start~end → 일수)
- ≤ 3일 → 기내용 캐리어·옷 최소 [출발 전·이동·편의] (선택)
- 4~7일 → 위탁 캐리어·옷 (일수−1)벌 + 세탁 1회 [출발 전·숙소·편의] (권장)

## 9. 귀국 정리 (time_axis=귀국)
- 항상 → 휴대용 수하물 저울·캐리어 무게 점검 [귀국·이동·필수] (권장) "초과 수하물 요금 방지"
